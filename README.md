# 正则表达式收集
- 邮箱验证 
```  /^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$/ ```
- 手机号验证
``` /^(13\d|14[579]|15[^4\D]|17[^49\D]|18\d)\d{8}$/ ```
- 固定电话验证
``` /^\d{3}-\d{7,8}|\d{4}-\d{7,8}$/ ```
- 邮编验证
``` /[1-9]\d{5}(?!\d)$/ ```
- 正整数验证 0-12
``` /^([0-9]|(1[0-2]))$/ ```
- 大于0的整数和小数
``` /^(0\.[1-9]\d*|[1-9]\d*(\.\d+)?)$/ ```
